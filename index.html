Hi, my name is Hakan and I am currently a third-year student in the Computer Engineering program at Marmara University. My primary area of interest is iOS development, and I have a strong passion for creating innovative and user-friendly applications for Apple devices.

Portfolio

  • iDeliver

    iDeliver is a mobile app that allows users to order groceries for delivery straight to their door. As the lead developer on this project, I used Swift programming language and the UIKit framework to create a user-friendly interface. I also implemented the MVVM (Model-View-ViewModel) architecture and the Coordinator pattern to ensure a clean and maintainable codebase.

    In addition to the frontend, I also built the backend of the app using Node.js and MongoDB. This involved designing a database schema, implementing API endpoints, and ensuring secure data transmission.

    With iDeliver, users can browse a selection of grocery items, add them to their cart, and check out with just a few clicks. The app also includes features such as real-time order tracking and push notifications to keep users informed about their delivery status.

  • iCoffee

    iCoffee is a mobile app that allows users to order coffee to their house. The app was developed using Swift programming language and SwiftUI framework, following the MVVM architecture pattern. The backend was written in Node.js and uses MySQL database to store user data.

  • ARFace

    ARFace is an iOS app that lets users apply fun and interactive face filters using augmented reality technology. Built using Swift programming language and a variety of powerful frameworks including UIKit, ARKit, SceneKit, and Firebase, this app is both innovative and user-friendly.
